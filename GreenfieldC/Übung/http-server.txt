Welche Methode wäre in welchem Anwendungsfall sinnvoll

GET: Um Daten abzurufen
POST: Daten an den SErver zu senden
DELETE: DAten auf dem Server zu löschen
PATCH: Daten teilweise auf dem SErver abzuändern
PUT: Daten auf dem Server ersetzen
HEAD: Damit fragt man ab, ob etwas vorhanden ist, ohne die eigentlich Information vom Server anzufordern (Ist das Kochbuch X da? Aber lade mir nicht das Kochbuch herunter!)
OPTIONS: Mit der Methode fragt man ab, welche Methoden insgesamt möglich sind, mit denen man Anfragen an Server schicken kann
