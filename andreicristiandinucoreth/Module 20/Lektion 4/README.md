# Setup Instructions

## Prerequisites

Node.js Version 16 or higher:

```bash
node --version
```

## Installation

```bash
npm install
```

This installs:
- `msw` – Mock Service Worker for HTTP mocking
- `jest` – Test runner

## Running Tests

```bash
# Run tests once
npm test

# Run tests in watch mode (re-runs on file changes)
npm test:watch
```

## Project Structure

- `newsAggregator.js` - News aggregator implementation (already implemented)
- `newsAggregator.test.js` - Test file with TODOs for you
