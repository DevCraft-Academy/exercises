## Aufgabe: Performanceanalyse der Erstellung von 1000 Elementen https://codepen.io/DevCraft/pen/NWVbNMb

Auffälligkeiten: Bei mir hat die Seite recht schnell geladen. 
Im Performance Tab konnte ich keine Bottlenecks feststellen.
Die Reflows und Repaints gingen recht schnell.
Trotzdem macht es Sinn erst alle Elemente in eine Variable zu packen und am Schluss mit einem Repaint zu rendern. Das geht mit createDocumentFragment():

document.addEventListener("DOMContentLoaded", () => {
    const startButton = document.getElementById("startButton");
    const listContainer = document.getElementById("listContainer");

    startButton.addEventListener("click", () => {
        startButton.setAttribute("disabled", "true");
        listContainer.innerHTML = "";

        const fragment = document.createDocumentFragment();

        for (let i = 0; i < 1000; i++) {
            const listItem = document.createElement("li");
            listItem.textContent = `Item ${i + 1}`;
            fragment.appendChild(listItem);
        }

        listContainer.appendChild(fragment); // Einmaliger DOM-Zugriff
    });
});

Anschließend war die Performance etwas besser. Je nach CPU Drosselung
